📚 StudyMatch – Backend
✅ Requisitos

- Python 3.8 ou superior
- pip instalado

🛠️ Instalação
Clone o repositório:

git clone https://github.com/motta1264/A2-Eng-Software.git

comandos iniciais:

- cd projeto_dos_crias
- pip install -r requirements.txt

▶️ Como rodar a aplicação

💻 macOS / Linux:

- export PYTHONPATH=.
- python app/main.py

🪟 Windows (CMD):

- set PYTHONPATH=.
- python app/main.py

🟦 Windows (PowerShell):

- $env:PYTHONPATH="."
- python app/main.py
